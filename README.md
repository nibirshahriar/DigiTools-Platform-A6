# 🌐 DigiTools

## 🚀 Description

DigiTools is a modern digital marketplace platform where users can explore, purchase, and manage premium digital tools, templates, and AI-powered resources in one place. It provides a clean UI, smooth user experience, and efficient cart management system.

---

## 🛠️ Technologies Used

* React.js
* Tailwind CSS
* DaisyUI
* React Icons
* React Toastify

---

## ✨ Features

* 🛒 Add to cart with duplicate prevention and toast notifications
* 🎨 Modern responsive UI with gradient design and clean layouts
* 📦 Dynamic product listing with pricing plans and feature highlights
